# Varad Jagtap - Developer Portfolio

Welcome to my personal developer portfolio! This project showcases my skills, experience, and the software projects I've built. 

## 🌐 Live Demo
Check out the live version here: **[https://varadraj-dev-portfolio.vercel.app/](https://varadraj-dev-portfolio.vercel.app/)**

## ✨ Features
- **Modern UI/UX**: Built with a sleek, responsive design using Tailwind CSS.
- **Accessible Components**: Utilizing Radix UI and shadcn/ui for highly accessible and customizable primitive components.
- **Smooth Animations**: Integrated micro-interactions and transitions for a premium user experience.
- **Contact Form**: Robust contact form functionalities using React Hook Form + Zod and EmailJS for seamless messaging.
- **Dark Mode Support**: First-class system & user-preference based theme switching using `next-themes`.
- **Fast Performance**: Powered by Vite for lightning-fast HMR and optimized production builds.

## 🛠️ Tech Stack
This project is built using modern web development technologies:
- **Framework:** React 18 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Architecture:** shadcn/ui & Radix UI
- **Routing:** React Router v6
- **Data Fetching:** TanStack React Query
- **Form Handling:** React Hook Form & Zod
- **Email Service:** EmailJS
- **Icons & Assets:** Lucide React & Lottie React

## 🚀 Getting Started Locally

To run this project on your local machine, follow these steps:

### Prerequisites
Make sure you have Node.js and an established package manager (`npm`, `yarn`, `pnpm`, or `bun`) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jagtapvarad17-stack/varadraj-dev-portfolio.git
   cd varadraj-dev-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or if using bun:
   bun install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add any necessary environment variables (e.g., EmailJS configuration):
   ```env
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **View in the browser:**
   Open the Local URL provided in your terminal (typically `http://localhost:8080`) to view the application.
